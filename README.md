# 🎓 Educational Institute Web Page

This is a preview of a responsive web application for managing an educational institute, built with **React** and **Material UI**.

---

## 🚀 Technologies Used

- React.js
- Material UI
- React Router
- Axios
- PHP for back
- MYSQL for databases
